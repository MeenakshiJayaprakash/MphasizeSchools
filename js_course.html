Variables and Data Types in JS

In JavaScript, variables are used to store data values. Variables in JavaScript are declared using the var, let, or const keyword followed by the variable name.

Here are the data types in JavaScript:

Primitive data types:

  1. string : Represents a sequence of characters, such as a word or a sentence. Example: "hello".
  2. number : Represents a numeric value. Example: 42.
  3. boolean : Represents a logical value that can be either true or false.
  4. null : Represents a null value, which means that the variable has no value at all.
  5. undefined : Represents an undefined value, which means that the variable has been declared but has not been assigned a value.
  6. symbol : Represents a unique identifier.

Object data type:

  1. object : Represents a collection of properties and methods. Objects are typically used to store structured data.

Operators in JS

In JavaScript, operators are used to perform actions on values and variables.

There are various types of operators in JavaScript, including:

  1. Arithmetic operators -
    1. It is used to perform arithmetic operations on values, such as addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).
  3. Assignment operators -
    1. It is used to assign values to variables, such as =, +=, -=, *=, /=, %=.
  4. Comparison operators -
    1. used to compare two values and return true or false, such as ==, ===, !=, !==, <, >, <=, >=.
  5. Logical operators -
    1. used to combine or negate expressions and return a boolean value, such as && (logical AND), || (logical OR), and ! (logical NOT).

Control Structurs in JS

Control structures are used to control the flow of code execution in JavaScript. There are several control structures in JavaScript, including:

  1. Conditional statements -
    1. used to execute code based on a condition. The most common conditional statements in JavaScript are if, else, and else if.
  2. Loops -
    1. used to execute a block of code repeatedly until a condition is met. The most common loops in JavaScript are for, while, and do-while.
  3. Switch statements -
    1. used to execute code based on different cases. The switch statement evaluates an expression and executes the code associated with the matching case.
  4. Break and continue statements -
    1. used to control the execution of loops. The break statement is used to exit a loop, while the continue statement is used to skip an iteration of the loop.

Functions

A JavaScript function is a block of code designed to perform a particular task.

A JavaScript function is executed when "something" invokes it (calls it).

Functions are an important feature of JavaScript that allow you to define a block of code that can be called and executed at any time during the program execution. A function can take input parameters, execute a block of code, and return a value if necessary.

How to write a function?

A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses ().Function names can contain letters, digits, underscores, and dollar signs (same rules as variables).

The parentheses may include parameter names separated by commas: (parameter1, parameter2, ...)

The code to be executed, by the function, is placed inside curly brackets: { }

Arrays and Objects

In JavaScript, arrays and objects are used to store and manipulate data.

Arrays:

Arrays are used to store a collection of elements of any data type in a contiguous memory location. Arrays in JavaScript are dynamic in nature, which means they can grow or shrink in size as per the need of the program.

How to define an array?

Using an array literal is the easiest way to create a JavaScript Array.

Syntax:

const array_name = [item1, item2, ...];

It is a common practice to declare arrays with the const keyword.

Objects:

You have already learned that JavaScript variables are containers for data values.

This code assigns a simple value (Mango) to a variable named fruit:

let fruit = "Mango";

Objects, on the other hand, are used to store a collection of key-value pairs. Each key in an object is unique, and it is associated with a value.

In JavaScript, objects can be created using object literals or constructor functions.

DOM Manipulation

DOM (Document Object Model) manipulation is an essential part of front-end web development. It involves manipulating the HTML document structure through JavaScript.

Here are some of the basic concepts of DOM manipulation in JavaScript:

  1. Accessing DOM elements : You can access DOM elements using various methods provided by JavaScript. The querySelector() and getElementById() methods are some of the commonly used methods.
  2. Modifying DOM elements : You can modify DOM elements by changing their properties such as innerHTML, textContent, style, className, etc.
  3. Creating new DOM elements :You can create new DOM elements using the createElement() method and append them to the existing DOM structure using theappendChild() method.
  4. Removing DOM elements : You can remove DOM elements using the remove() method.
  5. Event handling You can add event listeners to DOM elements to respond to user actions like clicking, hovering, scrolling, etc. using methods like addEventListener().
  6. Traversing the DOM You can traverse the DOM tree using properties like parentNode, nextSibling, previousSibling, firstChild, lastChild, etc.

These are some of the basic concepts of DOM manipulation in JavaScript. They are essential for creating dynamic and interactive web pages.
