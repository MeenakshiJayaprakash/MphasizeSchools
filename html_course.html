HTML Tags

HTML (Hypertext Markup Language) tags are used to mark up content on a web page. They are surrounded by angle brackets (< and >), and most come in pairs, with an opening tag and a closing tag. The opening tag tells the browser to start interpreting the tag, and the closing tag tells it to stop.


Some common HTML tags include:

  • <html>: This tag surrounds the entire HTML document.
  • <head>: This tag contains information about the page, such as its title and links to CSS stylesheets.
  • <body>: This tag contains the visible content of the page, such as text, images, and videos.
  • <p>: This tag defines a paragraph of text.
  • <a>: This tag creates a hyperlink to another web page or resource.
  • <img>: This tag displays an image on the page.
  • <ul> and <li>: These tags are used to create unordered lists.
  • <ol> and <li>: These tags are used to create ordered lists.
  • <div> and <span>: These tags are used to group and style other HTML elements.

HTML Attributes

Introduction

  1. All HTML elements can have attributes
  2. Attributes provide additional information about elements
  3. Attributes are always specified in the start tag
  4. Attributes usually come in name/value pairs like: name="value"

HTML attributes provide additional information about an HTML element. Attributes are added to the start tag of an HTML element, and consist of a name and a value, separated by an equals sign.

Here are some common HTML attributes:

  1. `class`: specifies one or more class names for an element, which can be used to style multiple elements with the same class.
  2. `id`: specifies a unique identifier for an element, which can be used to target that element with CSS or JavaScript.
  3. `src`: specifies the URL of the resource to be embedded, such as an image, video, or audio file.
  4. `href`: specifies the URL of the resource to be linked to, such as a web page or a file download.
  5. `title`: provides additional information about an element, often displayed as a tooltip when the user hovers over the element.

Attributes can be added to most HTML elements, including links, images, form elements, tables, and more.

HTML Structure and Formatting

HTML provides several formatting elements that allow you to modify the appearance of text and other elements on a web page. Here are some of the most common formatting elements:

  1. <b> and <strong>: both of these elements are used to make text bold. However, <strong> is considered to be more semantically meaningful and is used to indicate that the text is important or significant, while <b> is used purely for presentation purposes.
  2. <i> and <em>: both of these elements are used to italicize text. Like with <b> and <strong>, <em> is considered to be more semantically meaningful and is used to indicate emphasis, while <i> is used purely for presentation purposes.
  3. <u> : this element is used to underline text.
  4. <s> and <del> : both of these elements are used to strike through text. <s> is used to indicate that the text is no longer relevant, while <del> is used to indicate that the text has been deleted.
  5. <sup> and <sub> : these elements are used to create superscript and subscript text, respectively.
  6. <br> : this element creates a line break within a paragraph of text.
  7. <hr> : this element creates a horizontal rule (a line) to separate different sections of a web page.

These formatting elements can be used in combination with other HTML elements to create complex layouts and designs on a web page. It's important to use them appropriately and semantically, however, to ensure that your web page is accessible and can be understood by all users.
